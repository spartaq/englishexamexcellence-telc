## Architecture Changes

### New Routing Model
```
/telc/vocabulary              → Level selector view (B1/B2/C1 cards)
/telc/vocabulary/b1           → B1-only topic view
/telc/vocabulary/b2           → B2-only topic view
/telc/vocabulary/c1           → C1-only topic view
/telc/mywords                 → Global (unchanged)
```

### Data Flow
- **NO data structure changes** to VOCAB_HUB or VOCAB_LEVELS
- VOCAB_HUB remains source of truth for topic organization
- VocabHub component becomes **level-aware** via route param
- MyWords continues reading from VOCAB_HUB (no changes)

**Completion calculation update (use levelWords for current level only):**

const completionForLevel = (words) => {
  if (!words || words.length === 0) return 0;
  const mastered = words.filter(w => {
    const p = vocabProgress[w.term];
    return p && p.level >= 3;
  }).length;
  return Math.round((mastered / words.length) * 100);
