# Plan: Add Sales/Pricing Page (One-Time Payment)

**Goal**: Add a pricing/sales page to sell lifetime access passes, with integrated upgrade CTAs throughout the app.

**Payment Model**: One-time payment (not recurring subscription) - user pays once for lifetime access.

**Pricing Structure (Hungary Market)**:
- **Bronze**: Free, no registration required
- **Silver**: Free, but registration required (get more content unlocked)
- **Gold**: 9000 Ft (~€25) one-time - equivalent to 1 hour of English class

## 8. Payment Processor (Confirmed: Stripe + Clerk)

**Selected approach**: Stripe Checkout integrated with Clerk
- Stripe handles the payment processing
- Clerk handles user auth and can receive payment metadata
- Simple one-time payment links for V1 MVP

**MVP Flow**:
1. User clicks "Upgrade to Gold" 
2. Redirect to Stripe Checkout (one-time payment)
3. On success, user metadata in Clerk updated to show `hasLifetimeAccess: true`
4. Manual process initially, automate later when Stripe + Clerk integration is configured
